◆ GetGiocatoriRuolo()

public int fantacalcio.Giocatore.GetGiocatoriRuolo ( string  ruolo,
string  gruppo 
)

Ritorna il numero di giocatori in un certo ruolo in un certo gruppo.

Parametri
stringruolo: Il ruolo dei giocatori che si vogliono contare. Se viene inserito "tot" restituisce il numero di giocatori totale.
stringgruppo: Il gruppo dei giocatori tra cui si vuole contare: "r" = rosa, "t" = giocatori titolari, "a" = squadra attuale
intportieri: Il numero di portieri nel gruppo considerato
intdifensori: Il numero di difensori nel gruppo considerato
intattaccanti: Il numero di attaccanti nel gruppo considerato
intcentrocampisti: Il numero di centrocampisti nel gruppo considerato
Restituisce
int: Il metodo ritorna il numero di calciatori del ruolo specificato nel gruppo desiderato.

Viene controllato il valore della stringa gruppo. In base ad esso si decide in quale lista vanno contati i calciatori. Tramite un'istruzione switch dentro un ciclo foreach si incrementa la variabile corrispondente al ruolo del calciatore considerato nella lista desiderata. Poi, in base al ruolo desiderato, si ritorna il valore della variabile corrispondente. Se al posto del ruolo è stato inserito "tot", viene ritornato il numero di calciatori totali.

◆ GetStringSquadra()

public string fantacalcio.Giocatore.GetStringSquadra ( string  gruppo)

Il metodo ritorna una stringa contenente una lista di calciatori, che variano in base al valore della stringa ricevuta in ingresso, a cui è assegnato un indice pari alla loro posizione nella lista di appartenenza + 1.

Parametri
stringgruppo: Il gruppo di cui si vuole ottenere la stringa: "r" = rosa del giocatore, "t" = giocatori titolari, "a" = squadra attuale
stringstringa: La stringa contenente la lista di calciatori
List<Calciatore>daControllare: La lista di calciatori da cui si ricaverà la stringa
Restituisce
string: Il metodo ritorna una stringa contenente una lista in cui ogni riga contiene il nome e il ruolo di un calciatore e il suo indice.

Viene controllato il valore della stringa ricevuta come parametro, e in base ad esso si imposta il valore della lista daControllare a quello della lista desiderata. Con un ciclo for, che si ripete per un numero di volte pari al numero di calciatori presenti nella lista da controllare, viene scritta una nuova riga in cui si inserisce l'indice del calciatore, che corrisponde al numero dell'iterazione corrente + 1, il nome e il ruolo del giocatore nella posizione pari al numero dell'iterazione corrente nella lista da controllare. Viene infine ritornata la stringa.
